# 📖 Hello Rails Backend <a name="about-project"></a>

Hello Rails Back-end is exercise of creating a React front-end with a Rails back-end then connect them to display a random message.

## 🛠 Built With <a name="built-with"></a>

This app is build with Ruby on Rails and Postgresql.

### Tech Stack <a name="tech-stack"></a>

> <li><a href="https://www.ruby-lang.org/en/">Ruby</a></li>
> <li><a href="https://rubyonrails.org/">Ruby on Rails</a></li>
> <li><a href="https://www.postgresql.org/">Postgresql</a></li>

### Key Features <a name="key-features"></a>

## 💻 Getting Started <a name="getting-started"></a>

## To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Mac or PC
- Install Ruby
- Install Rails
- Install Postgresql

### Frontend

Here is the [Front-End](https://github.com/kambereBr/hello-react-front-end) of this app.

### Setup

- Clone this repository to your desired folder:

```sh
 git clone https://github.com/kambereBr/hello-rails-back-end.git
```

- Navigate into the cloned folder

```sh
 cd hello-rails-back-end
```

### Install

- Install this project with:

```sh
 bundle install
 npm install
```

- Set up the database
  Change the username and password of your postgres account in `config/database.yml`

```sh
 rails db:create db:migrate db:seed
```

- Start the server

```sh
 rails server
```
